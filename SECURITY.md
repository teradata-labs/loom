
  # Security Policy

  ## Supported Versions

  | Version | Supported          |
  | ------- | ------------------ |
  | 1.0.0   | :white_check_mark: |
  | < 1.0   | :x: |
  | < 1.0   | :x:                |

  ## Reporting a Vulnerability

  Please report security vulnerabilities to **via GitHub Issues (https://github.com/teradata-labs/loom/issues)**

  **Do not** open public GitHub issues for security vulnerabilities.

  Include:
  - Description of the vulnerability
  - Steps to reproduce
  - Affected versions
  - Potential impact

  We will respond within 48 hours and aim to resolve critical issues within 7 days.

  For general issues and questions, use [GitHub Issues](https://github.com/teradata-labs/loom/issues).
